# Roadmap

This roadmap keeps the project focused and maintainable.

## Current status

Personal Workspace is an early open-source template project. The current version provides:

- A documented folder structure.
- Reusable Markdown templates.
- A standard Python script to create local workspaces.
- Privacy-safe defaults for local personal work.

## Near-term priorities

- Add more templates for company comparison and meeting preparation.
- Add example workspaces with synthetic public data.
- Improve workspace naming for non-English company names.
- Add simple validation checks for generated workspaces.
- Document AI-assisted workflows for Codex, ChatGPT, Gemini, and NotebookLM.

## Medium-term ideas

- Add a small command-line index for created workspaces.
- Add export helpers for Markdown summaries.
- Add optional JSON metadata for each workspace.
- Add tests for the workspace creation script.

## Maintenance principles

- Keep the project easy to copy and audit.
- Prefer plain Markdown and standard Python.
- Avoid requiring accounts, paid services, or heavy dependencies.
- Keep private data out of the public repository.
