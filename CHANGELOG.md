# Changelog

All notable changes to this project will be documented here.

## v0.1.0 - 2026-05-31

- Initialized the public repository.
- Added README, MIT License, and contribution guide.
- Added workspace creation script.
- Added `--list-types` support to show available workspace types.
- Added company research, AI-assisted review, and job opportunity review workflows.
- Added reusable templates for company profiles, company comparisons, job opportunity reviews, and interview preparation.
- Added synthetic examples for company workspaces and company comparisons.
- Added tests and a GitHub Actions workflow.
- Added privacy-focused `.gitignore` rules for local workspaces.

## 2026-05-31
