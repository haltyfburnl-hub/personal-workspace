# Code of Conduct

This project aims to be practical, respectful, and useful.

## Expected behavior

- Be respectful and specific.
- Keep discussions focused on improving the project.
- Use examples that are public, synthetic, or properly anonymized.
- Avoid sharing private personal, financial, medical, legal, or confidential business information.

## Unacceptable behavior

- Harassment or personal attacks.
- Publishing someone else's private information.
- Adding confidential company material without permission.
- Submitting examples that expose credentials, contracts, resumes, or identity documents.

## Enforcement

Maintainers may edit, close, or remove issues and pull requests that violate these expectations.
