# AI-Assisted Review Workflow

This workflow helps users keep AI-assisted research auditable.

## 1. Start with sources

Put source notes in `sources/`. Keep each source separate where possible.

## 2. Ask focused questions

Use short prompts that point to one task at a time:

- Summarize the company in 10 bullet points.
- Identify risks and missing information.
- Draft interview questions based only on the source notes.
- Compare the opportunity against my written criteria.

## 3. Mark assumptions

When using AI output, separate:

- Source-backed facts.
- AI-generated summaries.
- Personal judgment.
- Items requiring verification.

## 4. Review before sharing

Before moving any file into `final/`, check:

- Does it contain private data?
- Are factual claims traceable to sources?
- Are assumptions clearly marked?
- Is the recommendation practical and specific?

## 5. Keep final outputs concise

A final output should be useful to a real decision, not just a long generated report.
