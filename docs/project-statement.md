# Project Statement

Personal Workspace exists to make personal research and AI-assisted work easier to audit, repeat, and improve.

Many people use AI assistants for serious work, but the surrounding materials often stay scattered across browser tabs, local folders, chat histories, and cloud documents. This makes it hard to know which sources were used, which conclusions are stable, and which drafts are ready to share.

This project provides a simple open-source structure for that workflow:

1. Keep source material separate from analysis.
2. Use consistent templates for company, role, and project research.
3. Record decisions and assumptions explicitly.
4. Keep private work out of the public repository.
5. Make AI-assisted output easier to review before it is used.

## Target users

- People preparing for interviews or career transitions.
- Operators and managers comparing business opportunities.
- Knowledge workers who need structured research folders.
- AI tool users who want a cleaner way to organize sources, prompts, drafts, and final notes.

## Non-goals

This project is not a resume builder, job board, CRM, or private document store. It is a small workspace template and workflow kit.

## Why open source

The project is intentionally plain Markdown and Python so that it can be copied, reviewed, changed, and reused without a paid platform or complex setup.
